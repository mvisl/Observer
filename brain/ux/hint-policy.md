# Hint Policy

Default hint mode: quiet.

Rules:

- no popups during active typing;
- no more than one surfaced hint per configured interval;
- all hints are logged as candidates;
- visible widget hints must be short;
- research digest is preferred over repeated interruptions.
- adaptive environment actions, such as music switching, start as suggestions until explicitly opted in.
- widget status should express second-level work interpretation, not raw sensor facts.

Good hint:

> Context is switching a lot. Collect context before asking another model.

Bad hint:

> You are distracted and should focus.
