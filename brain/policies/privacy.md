# Privacy Policy For Synced Brain

The synced brain is allowed to contain reusable product intelligence, not private user history.

Allowed:

- detector logic and thresholds;
- prompt templates;
- redaction patterns;
- UX copy;
- architecture decisions;
- anonymized examples;
- tests based on fake data.

Not allowed:

- real raw events;
- real OCR text;
- real screen titles from private work;
- camera frames;
- face landmarks from real sessions;
- access tokens, keys, passwords, emails, private client data.

External LLM requests must be generated from local summaries or manually approved exports.
