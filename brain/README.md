# Observer Brain

This folder is the portable, syncable part of Observer.

Safe to keep in a private GitHub repository:

- product principles;
- event schemas;
- detector definitions;
- hint policy;
- redaction rules;
- prompt templates;
- research digest templates;
- non-sensitive default settings.

Do not store here:

- `observer.sqlite`;
- screenshots;
- OCR dumps from real work;
- camera frames or derived per-second personal logs;
- private app/window titles;
- exported real context packs unless manually sanitized.

The local macOS app should treat this folder as shared product memory, not as user telemetry.
